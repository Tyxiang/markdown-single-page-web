## 3. Config

Use `## CONFIG` in the markdown file to add elements for the page:

| Markdown         | Web                | Other            |
| ---------------- | ------------------ | ---------------- |
| `### title`      | html head title    | will overwrite # |
| `### icon`       | html head icon     |                  |
| `### keywords`   | html head keywords |                  |
| `### logo`       | header logo        |                  |
| `### header-nav` | header nav         |                  |
| `### footer-nav` | footer nav         |                  |
| `### ending`     | footer ending      |                  |
